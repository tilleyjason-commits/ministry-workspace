## How to Use These Resources

### In sermon prep
When preparing a sermon or Bible study on a passage:
1. Open `commentaries/bible_exposition_commentary.md` for the book outline and key themes
2. Check `commentaries/expositors_ephesians_philemon.md` for detailed verse-by-verse commentary (Pauline epistles)
3. Use `commentaries/new_testament.md` for additional NT insight
4. Reference `preaching/challenge_of_preaching.md` for sermon structure and delivery principles
5. Use `tools/treasury_scripture_knowledge.md` for cross-references

### In this session
You can `read_file` any of these reference files directly. For large files, use `offset` and `limit` to navigate by section.

### Skill reference
Load the `ministry-bible-resources` skill to get guidance on which reference to use for any given passage or task.
